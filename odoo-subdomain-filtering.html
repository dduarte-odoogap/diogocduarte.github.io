Odoo Subdomain Database Filtering

Wed, 29 Oct 2014

After searching a lot i found that this is really possible to achieve, and it’s quite simple indeed just place this switch in server:

openerp-server -c configfile.conf --db-filter ^%d$

Note that this will not work if you place it in the configuration file.
