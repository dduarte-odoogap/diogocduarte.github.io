Docker Getting Started Commands

Sun, 23 Aug 2015

LXC provides the best available option for managing containers, community suported and base for lots of interesting projects out there. Docker used it on the first versions but now they handle directly with libcontainer.

After lots of troubles with wrong kernel versions that don’t support some features like unprivileged containers that need >= 3.12 and lxc-attach needs >= 3.8, i have compiled some startup info and commands.

# flush dnsmask cache
stop lxc-net
echo "" >/var/lib/misc/dnsmasq.lxcbr0.leases # erase current leases
